# E-COMMERCE-RFM-ANALYSIS-SQL
Cleaning and Segmenting 412,000+ rows of transaction data using MySQL to identify customer loyalty tiers.
Project Overview
I analyzed a dataset of *412,711 rows* to help an e-commerce business understand customer behavior. Using MySQL, I performed data cleaning and built an *RFM (Recency, Frequency, Monetary)* model to segment customers.

 Skills Demonstrated
* *Data Cleaning:* Handled duplicates, formatted dates, and trimmed text using TRIM, SUBSTRING, and REPLACE.
* *Advanced SQL:* Utilized CTEs, Window Functions (NTILE), and Aggregate Functions.
* *Business Logic:* Created segments (Champions, Loyalists, At-Risk) to drive marketing decisions.

  Key Queries
1. *The Cleaning Phase:* Standardized country names and converted date strings.
2. *RFM Scoring:* Assigned 1-5 scores to every customer based on their value.
3. *Market Basket Analysis:* Identified the top-selling product combinations.
